RV 2020 Tutorial Materials

This page holds all materials used for the tutorial The VyPR Framework for Performance Analysis of Python Web Services at the International Conference on Runtime Verification 2020.

Analysis Environment Demonstration

As a point of entry for analysing data recorded by VyPR at runtime, we provide a web-based analysis environment. The documentation for this can be found here.

Here, we provide a self-contained instance of the analysis environment used in our tutorial. You can get all the necessary code and set things up by running

curl -L http://cern.ch/vypr/setup-vypr-web-tool -O && source setup-vypr-web-tool
and then launch the tool with (in the same session) 
launch-vypr-web-tool

Visualisation

In order to allow people to see what's happening when VyPR instruments and monitors a program, we provide a visualisation tool. You can download this and set it up by running

curl -L http://cern.ch/vypr/setup-vypr-visualisation -o setup-vypr-visualisation && source setup-vypr-visualisation
and then launch it with 
launch-vypr-vis
